⚙️ Cài đặt lần đầu
Nhập URL của Google Apps Script Web App để kết nối với Google Sheets của bạn.
1
Mở script.google.com → New project
2
Dán nội dung file Code.gs vào, chọn đúng Spreadsheet
3
Bấm Deploy → New deployment → Web App
Execute as: Me · Who has access: Anyone
4
Copy URL Web App và dán vào ô bên dưới

URL được lưu trong trình duyệt, không gửi đi đâu khác.
