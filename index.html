⚙️ Cài đặt kết nối
Nhập URL Google Apps Script Web App để kết nối với Google Sheets.
1
Mở script.google.com → chọn project
2
Dán nội dung Code.gs → Save
3
Deploy → New deployment → Web App
Execute as: Me · Access: Anyone
4
Copy URL và dán bên dưới

URL lưu trong trình duyệt, không gửi đi đâu.
