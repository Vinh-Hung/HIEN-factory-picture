⚙️ Cài đặt kết nối
Nhập URL Google Apps Script Web App để kết nối với Google Sheets của bạn.
1
Mở script.google.com → chọn project của bạn
2
Dán nội dung file Code.gs → Save
3
Deploy → New deployment → Web App
Execute as: Me · Who has access: Anyone
4
Copy URL và dán bên dưới

URL lưu trong trình duyệt, không gửi đi đâu khác.
